# 最便宜航班查询工具 (find-cheapest-flight)

一个功能强大的 trip.com 航班搜索工具，支持普通航班查询和**隐藏城市（skiplag）机票**检测。自动寻找中转点为目标城市的更便宜机票。

## 功能特性

### ✈️ 航班搜索
- 自动爬取 trip.com 搜索结果，解析全部航班信息
- 支持直飞、中转航班搜索，按价格升序排列
- 自动识别并过滤隐藏城市机票（skiplag）：购买到更远城市的机票，在中转点（目标城市）下机，通常比直飞便宜 30%-50%

### ⚡ 性能优化
- **并行搜索**：4条隐藏城市航线同时打开浏览器标签页导航，单次等待覆盖全部，执行时间从 15 分钟压缩到 3-5 分钟
- **自动解析**：支持最新 SSE 格式 API 响应和旧格式，无需手动处理 `data:` 前缀
- **URL 直达**：使用已验证的 `showfarefirst` URL 格式，避免 404 和 UI 点击降级
- **中文输出**：完整的中文化输出格式，包含航班分段详情、中转等待时间、价格等信息

## 安装和使用

### 系统要求
- Chrome/Chromium 浏览器
- Chrome DevTools MCP 服务
- Python 3.10+

### 使用方法
```bash
/find-cheapest-flight <出发城市> <到达城市> <日期>
```

#### 示例
```bash
/find-cheapest-flight 巴黎 北京 2026-05-21
```

## 输出格式示例
```
# ✈️  CDG → PEK   2026-05-21

## 直飞 / 普通中转
1. 🥇 U23814+HU428+HU7368  CDG→PEK  $394  总时长38h20m
   ├ U23814  CDG(巴黎戴高乐) 13:40 → MXP(米兰马尔彭萨) 15:55  [2h15m]
   ├ HU428  MXP(米兰马尔彭萨) 18:55 → CKG(重庆江北) 12:40+1  [11h45m，候机3h]
   └ HU7368 CKG(重庆江北) 15:00 → PEK(北京首都) 17:20  [2h20m，候机2h20m]

## 隐藏城市票（仅乘前段落地北京，放弃后续航段）
1. 🔥 CA934+ZH9148  CDG→PEK→CAN  $559
   ├ CA934  CDG(巴黎戴高乐) 20:20 → PEK(北京首都T3) 12:40+1  [落地即下机]
   └ ~~ZH9148 PEK→CAN 14:55~~  ← 放弃此段
   ⚠️ 警告：只能带随身行李；常旅客账户存在被航司处罚风险
```

## 隐藏城市机票风险提示
⚠️ **重要提醒**：
1. 只能携带随身行李，托运行李会直挂最终目的地
2. 航司可能会对频繁使用隐藏城市票的用户进行处罚（冻结常旅客账户等）
3. 机票不可退改
4. 建议偶尔使用，不要高频使用

## 技术实现
- **浏览器自动化**：基于 Chrome DevTools MCP 协议
- **API 兼容**：同时支持 2026 新版 `FlightListSearchSSE` API 和旧版 `batchSearch` API
- **并行架构**：多标签页同时加载，最大化搜索效率
- **数据解析**：自动识别 SSE 流格式和 JSON 格式
